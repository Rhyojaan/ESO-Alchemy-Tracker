# The Alchemist – Project Bible

## Vision

The Alchemist is not just an ESO database.

The goal is to create the best ESO Companion available.

Players should need to leave the app as little as possible.

Aldren is the heart of the application. Every profession and every collection exists to provide Aldren with information so he can give intelligent advice.

---

# Core Principles

1. Every release must leave the application in a working state.

2. New professions are released only when they are complete.

3. Simplicity before complexity.

4. The player should enter information once whenever possible.

5. Aldren should do the thinking.

# Aldren

Aldren is not simply an assistant.

He is the player's companion.

He studies the player's archive and offers recommendations rather than simply displaying information.
